# HALO Family Companion Web (Phase 2)

Not in Y1 scope. Scaffold retained for future implementation.

Purpose:
- Receive Medium and Hard alerts from paired HALO devices.
- Initiate WebRTC video call to the protected HALO user.
- Approve or deny Hard-level transfer lock (second factor password).

Run:
```bash
cd family_web
npm install
npm run dev
```
